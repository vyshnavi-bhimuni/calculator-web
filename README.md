# Calculator Web Application

## Overview
This is a simple calculator web application developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations through an interactive and user-friendly interface.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Responsive user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone this repository.
2. Open `index.html` in any web browser.
3. Start using the calculator.
